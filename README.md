# Calorie Tracker

A simple calorie tracking application that helps you monitor your daily food intake and reach your nutritional goals. (Web & Mobile)

## Features

- Add and organize meals, foods, and ingredients
- Track your daily calorie consumption
- Set and monitor calorie goals
- Build a personalized food database

## Tech Stack

- React
- Vite
- Tailwind CSS, DaisyUI
- Supabase

## Self-Hosting

Detailed setup instructions will be provided once the project has enough features implemented.
